# DevelopersHub Corporation - AI/ML Engineering Internship

## Task 1: Exploring and Visualizing a Simple Dataset

### Objective
Learn how to load, inspect, and visualize a dataset to understand data trends and distributions.

### Dataset Used
- **Iris Dataset** - A classic dataset containing 150 samples of iris flowers across 3 species (Setosa, Versicolor, Virginica) with 4 features: sepal length, sepal width, petal length, and petal width.

### Models Applied
- No predictive model was applied in this task.
- Focus was on **Exploratory Data Analysis (EDA)** using pandas, matplotlib, and seaborn.

### Key Results and Findings
- The dataset contains 150 rows and 5 columns with no missing values.
- Petal length and petal width are the most distinguishing features between species.
- Setosa is clearly separable from the other two species based on petal size.
- Versicolor and Virginica show some overlap in feature distributions.
- Visualizations used: Scatter plots, Histograms, and Box plots.
